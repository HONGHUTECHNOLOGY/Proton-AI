GLM-4-9B-chat 负责知识问答和长文本理解
GLM-4V-Flash 负责图像识别

ERNIE Speed 负责检索问题优化
Gemini-2.0-Flash 负责心理辅导和聊天
Gemini-1.5-Flash 负责语音识别和图像识别
Gemini-2.0-Flash-Lite 负责心理知识库检索问题优化
